# BookTracker Application

A comprehensive book management and social reading platform that allows users to:

- Browse and search a book catalog
- Manage personal libraries with reading status tracking
- Connect with friends and share reading experiences
- Receive personalized book recommendations
- Access admin dashboard for content management
- Generate reports on reading activity and book popularity

The application targets book enthusiasts who want to organize their reading habits and discover new books through social connections. It provides both individual library management and community features for sharing reading experiences.

## Key Features

- User authentication with JWT-based security
- Personal library management (want to read, currently reading, completed)
- Social networking (friend requests, recommendations)
- Admin panel for book and user management
- Responsive design for mobile and desktop
- Reporting and analytics capabilities
